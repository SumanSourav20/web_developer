Table creating in html

HOMEVIDEOSNEXT BACKEND

HTML tables allow web developers to arrange data into rows and columns.In order to creat a table in web page you have to use the <table> </table> tag. Tables are divided into rows by <tr> tag,table rows are divided into table columns(table data) by <td> tag . You can also use table header tag <th> instead of table data tag to define column names.
Here is an example:

OUTPUT:

Firstname Lastname age
Jill Smith 50
Eve Jackson 94

NOTE:The <td> elements are the data containers of the table.They can contain all sorts of HTML elements; text, images, lists, other tables, etc.
